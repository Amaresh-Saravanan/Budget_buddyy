## Packages
recharts | For dashboard charts (bar, pie, line)
framer-motion | For smooth page transitions and micro-interactions
date-fns | For date formatting in lists and charts
react-day-picker | For calendar component (dependency of shadcn calendar)
clsx | Utility for conditional classes
tailwind-merge | Utility for merging tailwind classes

## Notes
- Using Shadcn UI components for base, customized with specific neon/dark theme
- Auth flow uses /api/register, /api/login, /api/user, /api/logout
- Recharts used for visual analytics
- Framer Motion used for page entry animations and list layout changes
